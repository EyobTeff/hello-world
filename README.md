# Overview

As a software engineer, I am continually improving my programming skills by learning new tools and technologies. This project helped me practice setting up a development environment, using Git and GitHub for version control, and writing a simple JavaScript program.

This software is a basic "Hello World" application written in JavaScript. When executed, it prints "Hello World" to the console.

The purpose of creating this software was to verify that my development environment was correctly configured and to practice the complete software development workflow, including writing code, using Git, publishing to GitHub, and documenting the project.
